## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Check for package updates

```sh
npx npm-check-updates
```

### Install package updates

```sh
npx npm-check-updates -u
```
